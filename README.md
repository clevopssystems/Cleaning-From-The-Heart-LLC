# Cleaning From The Heart LLC Website

Production-style Next.js lead-generation website for a Seattle cleaning business.

## Tech Stack
- Next.js (App Router)
- React + TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Run Locally
1. `npm install`
2. `npm run dev`
3. Open `http://localhost:3000`

## Website Strategy
1. Build trust in first 5 seconds with local positioning, strong hero copy, and visible call/quote CTAs.
2. Reduce decision friction with clear service segmentation and dedicated service pages.
3. Use visual proof (before/after) to move hesitant users toward action.
4. Keep mobile conversion paths always visible using repeated CTA blocks and sticky mobile action bar.
5. Capture high-intent leads quickly with a short quote form plus direct phone fallback.

## Sitemap
- `/` Home
- `/about` About
- `/services` Services overview
- `/services/commercial-cleaning` Commercial Cleaning
- `/services/carpet-cleaning` Carpet Cleaning
- `/services/strip-and-wax` Strip & Wax
- `/gallery` Gallery / Before & After
- `/contact` Contact / Get Quote

## Homepage Wireframe (Section Order)
1. Hero (headline, local subheadline, quote + call CTAs)
2. Trust/Credibility (review placeholders + trust stats placeholders)
3. Services cards (3 core services + links)
4. Before/After visual proof
5. How It Works (3-step process)
6. Strong CTA close
7. FAQ
8. Service Area
9. Footer

## Copy Structure Summary
- Focuses on reliability, professionalism, easy scheduling, and local trust.
- Avoids fake claims and fake testimonials.
- Uses explicit placeholders where real business data is required.

## Service Page Content Model
Each service page includes:
1. Local intent headline (Seattle + service)
2. Problem/solution framing
3. Benefits list
4. Process steps
5. Cross-link to Gallery + Contact CTA

## Conversion Recommendations
1. Replace testimonial placeholders with real Google review excerpts and source links.
2. Replace placeholder stats with real trackable metrics.
3. Connect quote form to a CRM/webhook and add instant confirmation.
4. Use call tracking number and UTM capture on quote form.
5. Add sticky call/quote bar on desktop if analytics show low CTA clicks.

## SEO Recommendations
1. Keep unique titles/meta descriptions per page (already structured).
2. Expand service area content with dedicated neighborhood landing pages over time.
3. Add real project photos with descriptive alt text + filenames.
4. Publish ongoing local content (facility cleaning tips, Seattle-specific maintenance topics).
5. Track calls, form submissions, and page-level conversion rates in GA4/Search Console.

## Important Placeholders To Replace
- Phone number, response-time blocks, review text, business stats, and detailed brand story.
- Placeholder images in `public/images/` with real project media.
